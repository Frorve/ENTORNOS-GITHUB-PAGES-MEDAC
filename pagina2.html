Guía de creación y publicación de la página

GitHub es una plataforma popular para alojar proyectos de software y colaborar con otros desarrolladores. Una de las características más útiles de GitHub es la capacidad de crear repositorios y agregar colaboradores para trabajar en el proyecto. Para la creación de nuestra página web tendremos que seguir una serie de pasos:

Primero de todo, tendremos que crearnos una cuenta en la pagina de GitHub, tras crear nuestra cuenta tendremos que crear un nuevo repositorio "Create repository" al cual le daremos el nombre que queramos.


Tras crear el repositorio nos tendremos que ir a los ajustes de este y en la pestaña "Collaborators" tendremos que añadir el nombre del usuario que queremos añadir de colaborador en el campo de búsqueda, para que así sean colaboradores y tengan acceso a nuestro repositorio.
